# نظام إدارة ضبط العدوى - يداً بيد

تطبيق عربي احترافي لإدارة ضبط العدوى في المنشآت الصحية، مبني بـ React وVite، ومجهز للنشر عبر GitHub Pages.

الرابط المتوقع بعد تفعيل GitHub Pages:
https://phmustafa00220-svg.github.io/ipc/

## الميزات
- لوحة مدير لمتابعة الالتزام والتنبيهات العالية.
- إدارة المنشآت الصحية مثل مشفى ابن الوليد ومشفى حماه مع إمكانية إضافة منشآت أخرى.
- إدارة حسابات وأدوار: مدير النظام، مدير منشأة، مسؤول ضبط العدوى، عامل صحي.
- إرسال تقارير فورية من العاملين إلى لوحة المدير.
- قوائم تحقق ومقاييس التزام قابلة للتعديل.
- تصدير التقارير CSV والطباعة.
- واجهة عربية RTL متجاوبة مع الهاتف والكمبيوتر.

## التشغيل محلياً
```bash
npm install
npm run dev
```

## بناء نسخة الإنتاج
```bash
npm run build
npm run preview
```

## النشر
المشروع يحتوي على GitHub Actions workflow في:
`.github/workflows/deploy.yml`

عند رفع الكود إلى فرع `main`، سيبني GitHub التطبيق وينشره على GitHub Pages.

## ملاحظة تشغيلية
النسخة الحالية تحفظ البيانات داخل متصفح المستخدم باستخدام LocalStorage. هذا مناسب كبداية وتجربة تشغيلية سريعة. للنسخة المؤسسية الكاملة يمكن لاحقاً ربط قاعدة بيانات وحسابات تسجيل دخول حقيقية.
