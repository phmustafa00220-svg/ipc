# نظام إدارة ضبط العدوى - يداً بيد

تطبيق عربي متجاوب مبني باستخدام React وVite لإدارة:
- المنشآت الصحية ودرجات الامتثال
- الجولات والتقييمات
- الحوادث والمخاطر
- التدريب والتوعية
- المخزون الوقائي
- التقارير والتصدير والطباعة

## التشغيل
```bash
npm install
npm run dev
```
ثم افتح الرابط الذي يظهر في الطرفية.

## بناء نسخة الإنتاج
```bash
npm run build
npm run preview
```

## ملاحظات
- البيانات محفوظة محلياً داخل المتصفح باستخدام LocalStorage.
- لا يحتاج التطبيق إلى قاعدة بيانات أو خادم للعمل في هذه النسخة.
