# نسخة المحطة الرئيسية - HIHFAD IPC

هذه النسخة تعمل بنفس فكرة الأنظمة المحلية القوية مثل XClinic:

- يوجد سيرفر Node محلي يشغل التطبيق.
- تسجيل الدخول يتم عبر API حقيقي وليس فقط إخفاء أزرار في الواجهة.
- كلمات المرور تحفظ مشفرة في `data/ipc-db.json`.
- الجلسات تحفظ بتوكنات مؤقتة في `data/ipc-sessions.json`.
- أي طلب بيانات عبر `/api/state` يرفض إذا لم يكن المستخدم مسجلاً الدخول.
- يمكن تشغيل التطبيق على جهاز رئيسي ومشاركته داخل الشبكة المحلية.

## التشغيل

شغل الملف:

```bat
start-ipc.bat
```

ثم افتح:

```text
http://127.0.0.1:3035
```

## الدخول الأول

```text
admin / admin123
ipc.ibn / ipc123
hama.nurse / 123456
```

بعد الدخول كمدير، أنشئ حسابات المستخدمين من تبويب المستخدمين.

## فتحه للموبايلات داخل نفس الشبكة

1. شغل `Allow-LAN-Firewall.bat` كمسؤول مرة واحدة.
2. اعرف IP جهاز الكمبيوتر.
3. من الموبايل على نفس الواي فاي افتح:

```text
http://IP-ADDRESS:3035
```
